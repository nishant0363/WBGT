WBGT Calculations for Villages

WBGT calculations have been completed for 30 villages over a span of 6 years (2010 to 2015).

The calculated WBGT results are summarized in three CSV files for each year. The first csv file is, "WBGT_Year_In_Range_Day.csv", contains the number of hours when WBGT falls within specific ranges according to standard values. This data is provided at the daily level, covering all 365 days for each village.

The second CSV file is the, "WBGT_Year_In_Range_Month.csv", also contains the number of hours when WBGT falls within specific ranges, but it is aggregated at the monthly level. This file includes data for all 30 villages, with each village having 12 rows representing the months.
